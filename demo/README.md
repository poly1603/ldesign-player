# @ldesign/player Demo

这是 @ldesign/player 的完整演示应用，展示了所有核心功能。

## 🚀 启动演示

```bash
cd libraries/player/demo
pnpm install
pnpm dev
```

演示将自动在浏览器中打开（默认端口：3000）

## ✨ 功能演示

### 🎵 音频播放器
- ✅ 播放控制（播放/暂停/停止/上一首/下一首）
- ✅ 音量控制（音量调节/静音）
- ✅ 进度控制（拖拽跳转）
- ✅ 播放速率（0.5x - 2x）
- ✅ 循环模式（不循环/单曲/列表/随机）
- ✅ 播放列表（点击切换）

### ⚡ 高级音频功能
- ✅ 波形可视化（实时波形/频谱显示）
- ✅ 歌词同步（自动高亮/点击跳转）
- ✅ 均衡器预设（平坦/摇滚/流行/爵士/古典/重低音）
- ✅ 音效处理（淡入/淡出）

### 🎬 视频播放器
- ✅ 播放控制（播放/暂停/停止）
- ✅ 视频控制（全屏/画中画/截图）
- ✅ 音量控制（音量调节/静音）
- ✅ 播放速率（0.5x - 2x）
- ✅ 状态显示（播放状态/时间/分辨率）

## 📦 技术栈

- **Vite** - 快速的开发服务器
- **@ldesign/player** - 音视频播放器核心库
- **howler.js** - 音频引擎
- **Web Audio API** - 高级音频处理
- **HTML5 Video** - 视频播放

## 🎨 界面特点

- 现代化的渐变设计
- 响应式布局（支持移动端）
- 流畅的动画效果
- 直观的交互体验

## 📝 使用说明

1. **音频播放器**：点击播放列表中的歌曲开始播放
2. **波形可视化**：点击"显示波形"或"显示频谱"按钮启动
3. **歌词同步**：播放音乐时歌词会自动滚动和高亮
4. **均衡器**：选择不同的预设体验音效变化
5. **视频播放器**：点击播放按钮开始播放示例视频

## 🔧 自定义

你可以修改 `main.js` 中的配置来测试不同的功能：

```javascript
// 修改播放列表
const playlist = [
  { id: '1', title: '你的歌曲', src: 'your-audio.mp3' },
];

// 修改视频源
const videoPlayer = createVideoPlayer('#videoContainer', {
  src: 'your-video.mp4',
  poster: 'your-poster.jpg',
});
```
